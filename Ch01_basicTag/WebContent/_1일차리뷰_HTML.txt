


1. 메타태그
 - 헤드에 넣는것
 meta : 글의 정보에 관한 내용을 쓰는 태그(key, content)
 title : 문서의 제목
 base : url 붙여넣는 태그
 link : url 붙여넣는 태그
 style : CSS 붙여넣는 태그
 script : 자바스크립트 붙여넣는 태그
 
 
2. Text 관련 태그
 - body에 넣음
 font : size, color, face 3가지의 속성을 가진다.
 b : 글자를 굵게한다
 i : 글자를 기울게한다.
 strong 중요한 글자
 sup : 위 첨자
 sub : 아래첨자
 
 
3. 구조관련 태그
 h1~h6 : 제목지정(1~3을 주로 씀)
 p : 문단지정
 br : 줄바꿈
 div : 영역 지정
 ul : 순서없는 목록지정  + li 리스트
 ol : 순서있는 목록지정 + li 리스트
 a : 문서의 이동 (속성 href="")
 
 
주소지정방식

 절대주소 : url을 기준으로 지정한다.
 
 상대주소 : 현재의 위치를 기준으로
		상위계층 : ../
		동일계층 : ./ -> (생략가능)
		하위계층 : /
